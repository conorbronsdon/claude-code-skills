# Reconcile

Tripwire check for multi-session drift. Scans recent commits, state files, and SSOT cross-references for inconsistencies caused by parallel Claude Code sessions or worktree merges. Run with `/reconcile` after merging branches, after crashes, or whenever something feels off.

Canonical home is now [agent-workspace](https://github.com/conorbronsdon/agent-workspace); this copy tracks it.
